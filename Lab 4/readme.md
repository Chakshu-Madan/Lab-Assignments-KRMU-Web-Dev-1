# Prompt Quizzer – JavaScript Console Quiz Game

This project is a simple interactive quiz game created for **Web Dev I – Lab Assignment 4**.  
The quiz runs entirely inside the **browser console**, uses JavaScript prompts for input, and alerts for feedback.

## Features
- Predefined array of questions and answers
- Input processed using `toLowerCase()` and `trim()`
- Score tracking
- Immediate feedback for each question
- Final score displayed at the end
- No HTML required

## Files Included
- `quiz.js` – Contains the full quiz logic
- `README.md` – Project explanation and running instructions
